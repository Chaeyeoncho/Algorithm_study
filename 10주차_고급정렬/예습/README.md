# 사용 방법

1. 위 표에서 문제 번호 또는 제목을 클릭합니다.
2. 링크를 통해 백준 온라인 저지 사이트에 접속합니다.
3. 문제를 읽고 알고리즘을 구현한 후, 제출하여 결과를 확인합니다.
4. 구현된 코드를 폴더에 push 하면 됩니다.
